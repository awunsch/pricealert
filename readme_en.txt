Thank you for downloading Price Drop Alert Module.


Installation
===========
Standard installation process applies. Unpack module to /modules directory and install from back office


Impact on System
================
new database table is created: PREFIX_ph_pricealert


FAQ
===

Q: How many emails will be sent to users?
A: Only one email is send for each subscription. Any subsequent price changes will not trigger email.


Q: No email has been sent after I've dropped price
A: Please verify that email template exists for your language.
